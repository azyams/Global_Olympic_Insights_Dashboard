# Global_Olympic_Insights_Dashboard
Global Olympic Insights Dashboard is a Power BI project that analyses Olympic Games data to understand medal performance, athlete participation, country rankings, and sports trends. The project includes data cleaning, data modeling, DAX measures, and interactive visualisations to provide meaningful insights from Olympic data.
This repository contains the original raw dataset, the cleaned dataset used for analysis, and the Power BI dashboard file developed for data visualization and insight generation.
